# Chaoxin_Cheater 超星作弊器

## 功能简介
超星学习通刷课脚本，可自动回答问题、连续刷课

## 使用方法
1. 打开https://www.chaoxing.com/（学习通网页版）
2. 右上角登录
3. 登录后，鼠标移动到右上角头像与用户名处，出现的选项栏中选择“进入空间”
4. 左栏选择“课程”，然后点击需要刷的课程
5. 左栏选择“章节”，然后点击需要从这里开始刷课的章节
6. 按F12键（笔记本可能要同时按fn+F12），选择“控制台（Console）”
7. 本GitHub页面直接点击“Chaoxin_Cheater.js”，并复制全部代码
8. 在步骤6处的“控制台”粘贴并回车（部分浏览器会因为粘贴而弹出警告，此时只需按照警告的要求做即可解除限制，然后再次粘贴即可）

## 实现逻辑
1. 两次尝试能否直接拖到100%
2. 倍速自动选最高
3. 题目用遍历的方法找到答案
4. 自动连播，没有下一个视频则打印“刷课结束”
